About

This is a sample Flask app for a UW Python Class in 2013. The application is built using the flask microframework and mongodb running on mongohq.



Flask Flask is a microframework for Python based on Werkzeug, Jinja 2 and good intentions.


Mongodb MongoDB (from "humongous") is a scalable, high-performance, open source NoSQL database.


Mongoengine MongoEngine is a Document-Object Mapper (think ORM, but for document databases) for working with MongoDB from Python. It uses a simple declarative API, similar to the Django ORM
